Installation: Famous 6-minute install

  1. Need a web server or hosted account with PHP 5 and MySQL (see Wordpress.org for details) and a domain name
  2. Create a MySQL database to be used for your p2press installation
  3. Upload p2press to your web server. The web server docroot should be /path/to/p2press/
  4. Similar to a regular Wordprss install, copy wp-config-sample.php to wp-config.php and edit the database settings in hat file with your own database account information
  5. Visit the domain and follow the Wordpress install instructions. LEAVE the username as 'admin' but of course personalize the rest.
  6. After install you must make the directory /p2press/wp-content/plugins/p2press/cache writable
  7. In wordpress admin, remove all existing blogroll links.
  8. Change your nickname under Word press Admin->users. This what other p2press users will see you as.
  9. Inside the Wordpress admin go to plugins and activate the p2press plugin
  10. Add some blogroll links in Links->Add New to other p2press users.These are the people you follow. You should use their home page domain name as the web address in in the form. You must add their RSS url down below in the field entitled 'RSS Address" Make sense?
  11. Optionally, you should add the Blogroll widget to your side bar. This isn't strictly necessary but is a big help in bootstrapping your community.
